# Femina Privacy Policy & Terms

This repository contains the legal documents for the **Femina** Android application, hosted via GitHub Pages.

## How to Deploy to GitHub Pages

1. Create a new public repository on GitHub named `FeminaPrivacyPolicy`.
2. Push these files to the `main` branch.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, set the source to **Deploy from a branch** and select `main` (root folder).
5. Once deployed, your privacy policy will be available at: `https://[your-username].github.io/FeminaPrivacyPolicy/`

## Features
- **Bilingual:** Supports English and Bahasa Indonesia.
- **Responsive:** Works on mobile and desktop browsers.
- **Themed:** Matching Emerald & Cream aesthetic.
